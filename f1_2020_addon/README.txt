F1 2020 Add-on for Splunk
=========================

Add-on built to collect Codemasters F1 2020 game data via UDP packets then convert into JSON before sending to Splunk.

Leveraging code from -

1) https://pypi.org/project/f1-2020-telemetry/ to capture the data into python classes
2) https://github.com/rinatz/ctypes_json to convert python classes into JSON

Add-on logo is from https://www.onlinewebfonts.com/icon/467043

This is an add-on powered by the Splunk Add-on Builder.
